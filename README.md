# Dog-Training
I love dogs and wanted to build a fictitious site for dog training services. I included a form at the bottom for requesting information. It takes the information that is inputted, runs validations on it and if the information is good it composes a confirmation message at the bottom of page that includes the inputted information. If the email is not in a valid format it displays a message to input a valid email. If form field's are left blank a popup says please fill in this field.
