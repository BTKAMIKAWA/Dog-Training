# Dog-Training
I love dogs and wanted to build a fictitious site for dog training services. At the top of the webpage, there's a nav bar with links to the 3 sections of the webpage. Each section has an ID that is in the href of the corresponding link, telling it where to go when the link is pressed. Our services section was designed using components like styling. A card div was created to utilize the styling I liked, and allowed me to reuse the same styling three times. Once for each card div or service. Flexbox was used so that each card div would display side by side and so that I could give them each a grow, shrink, and basis property, which gives each card a starting size and tells them how much to grow or shrink when the page size changes. I also included a form at the bottom for requesting information. It takes the information that is input in the various fields, runs validations on it via JavaScript, and if the information is valid, it composes a confirmation message at the bottom of the page that includes the inputted information from each field. If the email is not in a valid format, it displays a message to input a valid email. If form fields are left blank, a pop-up says please fill in this field.
